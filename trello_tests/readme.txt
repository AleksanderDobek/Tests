1.Windows and browser Chrome should be up to date
2.Download and Install Python (with adding to PATH) https://www.python.org/downloads/
3.Download and Install PyCharm https://www.jetbrains.com/pycharm/download/#section=windows
4.Download ChromeDriver https://chromedriver.chromium.org/downloads (ChromeDriver version should be compatible with Chrome)
5.Create dir C:\TestFiles and copy there chromedriver.exe from downloaded zip
6.Lauch PyCharm, open trello_tests(downloaded from Git) as a project
7.Make sure You have up to date Project Interpreter configurated (File - Settings - Project:trello_tests - Project Interpreter),
 add it if it nessesary ..\trello_tests\venv\Scripts\python.exe
8.Install package selenium by adding '+' on Project Interpreter view, submit
9.Run tests
